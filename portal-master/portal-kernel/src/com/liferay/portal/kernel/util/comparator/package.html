This package defines the portal comparator classes.

Make use of comparators for:

  • Categories
  • Dates
  • Descriptions
  • Email addresses
  • IDs
  • Names
  • Priorities
  • Titles
  • Types
  • Weights

These comparators apply to the following entities:

  • Groups
  • Layouts
  • Organizations
  • Password policies
  • Portlets
  • Resources
  • Roles
  • Subscriptions
  • Users
  • User groups
